# Interview Agent Sparring Partner 🥊

An AI-driven conversational agent designed to act as a dynamic interview sparring partner. This tool helps candidates practice technical and behavioral interview questions in a realistic, unscripted environment to refine their responses and build confidence before the real thing.

## Features
* **Dynamic Questioning:** Adapts to user responses rather than following a rigid script.
* **Real-time Feedback:** (Planned) Provide immediate critiques on response structure and technical accuracy.
* **Contextual Memory:** Maintains the thread of the conversation for follow-up questions.

## Getting Started

### Prerequisites
* Mac OS or Linux environment recommended.
* Python 3.x

### Installation

1. Clone the repository:**
   ```bash
   git clone [https://github.com/kazuo-shimada/interview-agent-sparring-partner.git](https://github.com/kazuo-shimada/interview-agent-sparring-partner.git)
   cd interview-agent-sparring-partner
   
2. Set up the virtual environment:

   ```bash
   python3 -m venv sparring_env
   source sparring_env/bin/activate

3. Install dependencies:

   ```bash
   python3 -m pip install -r requirements.txt

Enjoy your technical interview agent!
